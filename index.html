📲 PWA 푸시 알림 설정

별도 앱 설치 없이 이 앱을 홈 화면에 추가하면 잠금 화면 알림을 받을 수 있습니다.
Cloudflare Workers 설정 완료 후 아래 버튼으로 테스트하세요.

📱 학생·학부모 안내
  1. 앱 접속 QR 스캔 (아래)
  2. 브라우저에서 "홈 화면에 추가" 선택
  3. 앱 실행 → 알림 허용
  4. 이후 공지 작성 시 자동 알림 수신
🔔 푸시 알림 테스트

Cloudflare Workers 설정 완료 후
테스트 알림을 전송해보세요.

📋 설정 순서: ① generate_vapid.py 실행 → 키 생성 ② cloudflare-worker.js 를 Cloudflare 대시보드에 배포 ③ index.html 의 VAPID_PUBLIC_KEY, PUSH_WORKER_URL 값 입력

🔥 Firebase 보안 규칙 설정 필요

공지사항이 학생/학부모에게 실시간 전달되려면 Firebase Console에서 아래 규칙을 설정해야 합니다.

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

설정 방법: Firebase Console → Firestore Database → Rules 탭 → 위 내용 붙여넣기 → 게시
